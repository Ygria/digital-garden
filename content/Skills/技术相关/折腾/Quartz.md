用于搭建Obisidian数字花园页面。
## 页面预览

已初始化后，在quartz文件夹执行：
```shell
npx quartz build --serve
```
进行预览。（8080端口）

## Github 管理
首次与Github仓库同步：

```shell
npx quartz sync --no-pull
```

后续可以省略no-pull

## 构建指令
```shell
npx quartz build
```
